# Kerobot - Panel Yakuza Tanaka

## Render

Build Command:
npm install

Start Command:
node index.js

## Variables de entorno

TOKEN
CLIENT_ID
GUILD_ID

## Comandos

/panelrobos
/alertarobos
/resetrobos

El bot incluye:
- Panel bonito con embed
- Botones
- Robos diarios, semanales y cada 2 semanas
- En robos 1/1 guarda quién lo hizo
- Alerta fija editable
- Web activa para Render gratis
